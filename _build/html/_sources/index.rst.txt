Sobre o Projeto
==========================

O projeto chama-se Servidor de Deaafios e foi criado pelo professor 
Rauk Ikeda como uma plataforma para envio de soluções de codigos dos 
alunos para posterior avaliação

Como Usar
==========================
Para usar o projeto, basta inserir os parametros de seu usuario em arquivo users.csv na raiz do projeto (src).
Ainda pode ser especificado se seu usuario sera ou nao um usuario administrador

Para começar a usar é só logar seu usuario na pagina principal /. Por default, o projeto roda na porta 80 da maquina.

Guia de Instalação
==========================

O projeto é feito construindo um servidor com a linguagem python e módulo flask.
Também é usado um DB em arquivo, o SQLite3

Além disso, o projeto depende das seguintes bibliotecas externas:

Certifique-se de possuir as seguintes dependências:

* Python 3.7 ou superior
* flask
